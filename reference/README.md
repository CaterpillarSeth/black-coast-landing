# Brand Book Reference

Place the Black Coast Estates brand book PDF here as `brand-book.pdf`.

The brand guidelines used for this landing page are:

## Colors
- Ivory: #FEFDFB
- Cream: #F8F6F1 (default surface)
- Champagne: #C5A96A (primary accent — never on light backgrounds)
- Brass: #9E8548 (deeper gold)
- Hunter: #4A6358 (muted warm green)
- Deep: #3A4F46 (body text on light)
- Sage: #95A89D (secondary text)

## Typography
- Headings: Playfair Display, Medium (500), italic for accent only
- Body: Lora, Regular, 8.5pt, 1.7 line-height
- UI/Labels: Raleway, Light to SemiBold
- Numbers/Stats: Outfit

## Tone
"Quiet luxury" — warm, natural, unhurried
Taglines: "Where deep-rooted nature meets quiet prestige." / "An invitation to own something that endures."
